# Roll_Stabilized

## WHY 
Model rockets roll during ascent due to aerodynamic asymmetries. 
Characterising aerodynamic properties of the rocket from flight 
data requires - as a first step - single-axis roll stabilisation.

## Hardware
- IMU: ICM20948,
- MCU: Teensy 4.1
- SD logging
- Servo (fin actuator)
